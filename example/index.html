    Options

    Name Default Options avaliable
    visible_items 1 Any Integer
    width css css - use the width of the carousel from css
    auto - the auto width is primarily of benefit for when you need a flexible layout, it handles the resizing of the window so the carousel continues to work
    Pager
    pager_type dots dots -
    list -
    pager_title    
    pager_position outside  
    pager_content number  
    pager_template   The pager template
    Callbacks
    callback_onsetup   Function that loads after the carousel has been setup
    callback_onprev   Function that triggers after the prev button is clicked
    callback_onnext   Function that triggers after the next button is clicked
